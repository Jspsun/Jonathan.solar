Internet Thoughts

Internet Thoughts is a web-based sentiment analysis engine that works as follows:

  1. The user types in any topic
  2. We make API calls for Twitter, Reddit and Tumblr related to the given topic
  3. We run sentiment anlysis on the fetched data to determine how the internet feels about the given topic

JavaScript, Node.js, CSS, HTML, EJS, Heroku, Twitter API, Reddit API, Tumblr API
