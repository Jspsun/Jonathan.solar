Internet Thoughts

Internet Thoughts is a web-based sentiment analysis engine that works as follows:

  1. The user types in any topic
  2. We make API calls for Twitter, Reddit and Tumblr related to the given topic
  3. We run sentiment anlysis on the fetched data to determine how the internet feels about the given topic

JavaScript, Node.js, CSS, HTML, EJS, Heroku, Twitter API, Reddit API, Tumblr API

Athena IOT Smarthome AI System

Y'know jarvis from Iron Man (the movies)? That's exactly what I set out to build. I'm running Athena on a raspberry pi with a smart mirror for the UI. It does everything from display the weather, show me my messages to turning on my lights and playing music for me. I even wrote an android app for when I'm out and about! Here's how it works:

  1. You tell Athena what to do
  2. Athena uses speech to text engines to process your input into text form
  3. Using a deep learning algorithm, Athena determines which scripts and commands you want it to run

Python, Google Natural Language Processing, Twilio API, Android Studio

Really Easy Bibliography

There's EasyBib and then there is ReallyEasyBib. EasyBib will help you source all the website you use for research. ReallyEasyBib on the otherhand just lets you type in a topic and generates a fully cited list of resources for you!

Imagine it's 3:00am and you have a paper on hamlet due tommorow and for some godforsaken reason, your teacher demands that you use 10 different web-sources. Seriously, it's 3:00am and you want to sleep. Most students would just google hamlet and shove the first 10 links (one by one) into EasyBib and pray that they get sourced right. Well ReallyEasyBibliography takes that and make it even easier. All you need to do is type in a subject and it generates the entire bibliography for you :)
P.S Please don't tell my high school english teacher

Python, PHP, CSS, JavaScript, JQuery, Google Scholar Search API
